# 🤖 **astribot_msgs** | Unified Data Structures for Control and Simulation

**astribot_msgs** repository provides unified data structure definitions for Astribot robot control and simulation.
**This repository is integrated as a submodule in [astribot_simulation](https://github.com/Astribot-Dev/astribot_simulation) and [astribot_sdk](https://github.com/Astribot-Dev/astribot_sdk/tree/ros1) projects, implementing message communication based on ROS1 Noetic.

---

## 📦 Features

- Provides ROS message definitions required for robot control and simulation
- Ensures consistency of data structures between simulation and actual control
- Supports multi-project reuse, easy to maintain and extend

---

## 🖥️ System Requirements

- **Operating System**: Ubuntu 20.04 LTS
- **ROS Version**: ROS Noetic

---

## 🚀 Quick Start

### 1. Environment Setup
- source <your_path>/noetic/setup.bash
- cd <your_path>/astribot_msgs
- catkin_make install

### 2. Load Environment
- source <your_path>/astribot_msgs/install/setup.bash
